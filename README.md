# Минимальное ядро с сетевой поддержкой и загрузчиком пакетов

## Структура проекта

```
kernel/
├── boot/           # Код загрузки (bootloader + kernel entry)
├── drivers/net/    # Сетевые драйверы
├── fs/             # Файловая система
├── libc/           # Минимальная стандартная библиотека
└── apps/           # Приложения/пакеты
include/            # Заголовочные файлы
```

## Требования

- **QEMU** - эмулятор для тестирования
- **NASM** - ассемблер для x86_64
- **GCC** - компилятор C (для частей ядра)
- **GNU LD** - линковщик
- **Git** - для загрузки пакетов

## Быстрый старт

```bash
# Сборка ядра
make build

# Запуск в QEMU с сетью
make run

# Загрузка пакета из GitHub
# (внутри ядра через консоль)
pkg install username/repo
```

## Архитектура

### Bootloader (16-bit Real Mode → 32-bit Protected Mode → 64-bit Long Mode)
- `boot/boot.asm` - загрузчик сектора
- `boot/kernel_entry.asm` - входная точка ядра

### Ядро (C + Assembly)
- Инициализация GDT, IDT, PIC
- Инициализация сети (PCI, драйвер сетевой карты)
- Простой TCP/IP стек
- Загрузчик пакетов через HTTP/HTTPS

### Пакетный менеджер
- Скачивание бинарных пакетов из GitHub Releases
- Простая файловая система для хранения приложений
